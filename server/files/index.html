Retirement Calculator

I built this as a portfolio site with vanillaJS. The source of some the calculations comes from Charles Schwab, namely the matrix in the center of the worksheet.

Something that has always bothered me is the discounting of dollars in all of these calculations and the present value of money vs discounted dollars on the future Social Security check you're going to receive. This isn't a better mousetrap but an intellectual exercise for me to build a website and to see what this math is all about.

Disclaimer: If you really need professional financial advice, seek out an actual financial management professional. Don't use these calculations.

This model assumes a 3% inflation rate for discounting.

None of this data is persisted in a database so feel free to click away and try out your scenarios.
